# fish-macos Security Policy

## LLM-Generated Security Report Policy

Absolutely no security reports will be accepted that have been generated by LLM
agents.

## Supported Versions

This fish plug-in is supported on a rolling basis. Only the most recent version
will receive security updates.

This plugin is supported only on Fish 4 or later.

## Reporting a Vulnerability

Create a [private vulnerability report][advisory] with GitHub.

[advisory]: https://github.com/halostatue/fish-macos/security/advisories/new
